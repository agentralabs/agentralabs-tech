# Feedback and Community

Use this page to report issues, request features, and share runtime outcomes.

## Where to send feedback

- Workspace and orchestration issues: [agentralabs-tech](https://github.com/agentralabs/agentralabs-tech/issues)
- AgenticMemory issues: [agentic-memory](https://github.com/agentralabs/agentic-memory/issues)
- AgenticCodebase issues: [agentic-codebase](https://github.com/agentralabs/agentic-codebase/issues)
- AgenticVision issues: [agentic-vision](https://github.com/agentralabs/agentic-vision/issues)

## What to include

- OS and architecture (for example: macOS arm64, Ubuntu x86_64).
- Installed profile (`desktop`, `terminal`, or `server`).
- Exact command that failed.
- Full error output.
- Whether `.amem`, `.acb`, `.avis` were local or server-synced.

## Before filing

- Run `agentra doctor`.
- If in server mode, run `agentra server preflight --strict`.
- Review the troubleshooting matrix in [Workspace How-To](/docs/workspace-how-to#troubleshooting-matrix).

## Community expectations

- Keep reports reproducible.
- Avoid sharing secrets, tokens, or private datasets.
- Prefer one issue per distinct defect.
